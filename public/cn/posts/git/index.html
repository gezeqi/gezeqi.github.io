Git

安装

下图为MacOS版本的Git官方网址的下载页面，可以看到有许多方法可以下载安装Git。

  • 方式1:如果需要在Mac上进行编程开发，最简单的方式便是通过下载Xcode来安装Git。
  • 方式2:如果想要自定义安装，推荐使用Homebrew来安装Git。

配置

在终端中输入如下命令以配置用户名与邮箱：

git config --global user.name "gezeqi"
git config --global user.email "ge051799qi@gmail.com"

在终端中输入如下命令以查看配置列表：

git config --list

生成SSH密钥

在终端中输入如下命令以生成SSH密钥：

ssh-keygen -t ed25519 -C ge051799qi@gmail.com

如果不想过于复杂的操作，只需要不断回车完成即可。

查看公钥

默认在用户名下生成一个.ssh文件夹，需要首先按command+shift+.以显示隐藏文件。

Github中添加公钥

点击右上角头像–>Settings–>SSH and GPG keys，在SSH keys中点击"New SSH key"。

在页面中输入对应内容，点击确认即完成配置。

  • Title自取，
  • Key type不变，
  • Key则输入查看的公钥内容，

后续在终端中使用Git将会自动认证，而无需每次在弹出窗口中进行认证。

注意

Mac上Git已经设置了SSH连接，但是使用git push时仍然需要输入用户名和密码

这可能是因为Git仓库远程URL仍然是HTTPS格式，Git会尝试使用HTTPS进行认证，而不是SSH。

检查当前远程URL

git remote -v

得到如下输出，这代表使用的是HTTPS格式。

origin  https://github.com/your-username/your-repo.git (fetch)
origin  https://github.com/your-username/your-repo.git (push)

更改远程URL为SSH

git remote set-url origin git@github.com:your-username/your-repo.git

完成后可再次检查远程URL
